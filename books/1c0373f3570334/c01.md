# 1. この章でやること
この章では、本書の対象読者と読み進め方を整理します。

また、目的別に「どの章を読めばよいか」の目安も提示します。
ユーザー環境だけ、開発環境だけ、とちいさく始めてみたい場合は参考にしてください。


# 2. 誰のための本か
本書は、すでに Homebrew などで環境構築を行っているユーザーを想定しています。

**主に Mac ユーザー を対象に解説**します。
基本的な仕組みは Linux（WSL2 含む）でも共通ですが、Mac 特有の設定（nix-darwin）を含みます。

**Linux ユーザーの方は、nix-darwin と書かれた箇所は読み飛ばしてください**。


# 3. 本書の進め方
本書では、いきなりすべてを Nix 化するのではなく、**小さく始めて、徐々に置き換える**方針で進めます。

**以下を全て実行すると Mac の環境全体を管理できます**が、最初から実施するのは**学習コストが大きい**かと思います。

1. home-manager でユーザー環境（CLI ツールなど）を管理する
2. home-manager で管理できないツール（Homebrew にしかないツール）を nix-darwin で補完する
3. nix-darwin で Mac のシステム設定も管理する
4. flakes devShell で開発環境を分離・再現する

そのため、興味がある内容から触ってみることをおすすめします。
以下にモデルケースを示します。


# 4. 目的別の読み進め方

## 4.2 開発環境を管理してみたい

## 4.3 Mac 全体を Nix で管理したい
c02 → c03 → c04 → c05 → c06 → c07 → c08 → c09 → c10 → c11 → c12 → c13 → c14 → c15

## 4.4 筆者のおすすめ
c02 → c03 → c04 → c05 → c06 → c07 → c08 → c09 → c10
→ c16 → c17


:::message
次章で解説しますが、既存の管理ツール（Homebrew 等）と Nix は併用できます。

ですので、ユーザー環境だけ / 開発環境だけ Nix を試すことも可能です。
まずは興味がある使い方で Nix に触れてみてください。

そして、Nix にハマったら徐々に Nix 管理の幅を広げていくと良いと思います。
:::
