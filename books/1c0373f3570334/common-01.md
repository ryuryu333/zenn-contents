# 1. 誰のための本か
Nix は使っておらず、Homebrew や mise でパッケージ管理している方を想定しています。
「**Nix に興味を抱いたけど、具体的にどう使い始めればいいのか？**」と感じている方に向けた本です。

**本書では、ユーザー環境や開発環境の管理を徐々に Nix に移していく流れを体系的に整理・解説します**。

:::message
対象 OS は Windows（WSL）、macOS、Linux です。
本書のコマンド・コード例は Windows11（WSL2 Ubuntu 22.04.5 LTS）、MacBook Pro M1 にて動作検証しています。
:::


# 2. 読み進め方
Nix は高い再現性が特徴のパッケージマネージャーです。
代表的な用途は、ユーザー環境の管理、開発環境の管理、の 2 つです。

**それぞれ独立した方法でパッケージが管理されるため、ユーザー環境だけ Nix を試してみる、といったことも可能です**。

まずは自身の興味がある領域（ユーザー環境 or 開発環境）で Nix を試し、気に入ったら Nix で管理する幅を広げていくと良いかと思います。

----

本書は五部構成になっています。

1. Nix について

第二部〜第四部は興味がある所からお読みください。

知っていると便利な知識を雑多にまとめています。
