# 1. この章でやること
この章では **home-manager の基本的な使い方**を体験します。  
まずはツールを 1 つだけ導入し、Homebrew との違いを確認します。

# 2. 最小構成で動かす
## 2.1 home.nix にツールを追加
ここでは例として **git** を追加します。  
`~/.config/home-manager/home.nix` を開き、`home.packages` に追記します。

```nix:~/.config/home-manager/home.nix
home.packages = with pkgs; [
  git
];
```

## 2.2 反映する
以下を実行します。

```bash:Bash
home-manager switch
```

# 3. 使えるようになったか確認
```bash:Bash
git --version
```

バージョンが表示されれば成功です。

# 4. Homebrew と競合しないの？
結論として、**同じツールが入っていても問題ありません**。  
実際に使われるのは **PATH の先にある方**です。

確認したい場合は `which git` でどちらが使われているか分かります。

```bash:Bash
which git
```

# 5. この時点では削除しない
この章の目的は **home-manager の基本動作を体験すること**です。  
Homebrew の git はまだ削除しなくて大丈夫です。
