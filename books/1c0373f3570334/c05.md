---
title: "Homebrew から home-manager へツールを移行"
---

# 1. この章でやること
この章では、**Homebrew で入れていたツールを home-manager に移し始める**ことを目標にします。  
最初は **git だけ**を移行し、設定ファイルも合わせて管理します。

>段階的に進めるため、既存の Homebrew と **併用**します。

# 2. 最終形のイメージ
home-manager の設定は `~/.config/home-manager` に集約します。  
この章のゴールは以下のような構成です。

```
~/.config/home-manager/
├── flake.lock
├── flake.nix
├── git
│   └── .gitconfig
├── home.nix
└── zsh
    └── .zshrc
```

# 3. git を home-manager で管理する
## 3.1 home.nix に git を追加
`home.nix` に git を追加します。  
（この時点では Homebrew と併用しているので、削除しなくて大丈夫です）

```nix:~/.config/home-manager/home.nix
home.packages = with pkgs; [
  git
];
```

反映します。

```bash:Bash
home-manager switch
```

## 3.2 競合しないの？
Homebrew と home-manager で **同じツールが入る状態**になりますが、  
基本的には **先に PATH にある方が使われる**だけです。

まずは home-manager 側の git が優先される状態になれば OK です。  
後で Homebrew 版を削除すれば、よりスッキリします。

# 4. 設定ファイルを一緒に管理する
## 4.1 .gitconfig を管理する
`~/.config/home-manager/git/.gitconfig` を用意し、`home.nix` でリンクします。

```nix:~/.config/home-manager/home.nix
home.file = {
  ".gitconfig".source = ./git/.gitconfig;
};
```

反映します。

```bash:Bash
home-manager switch
```

# 5. シェル設定も管理する
`~/.config/home-manager/zsh/.zshrc` を用意し、同じようにリンクします。

```nix:~/.config/home-manager/home.nix
home.file = {
  ".zshrc".source = ./zsh/.zshrc;
};
```

反映します。

```bash:Bash
home-manager switch
```

# 6. 以降は順次移行する
この章では **git と設定ファイルだけ**を移行しました。  
同じ手順で、Homebrew で入れていたツールを **ひとつずつ移行**していきます。

次章以降で、移行を進める上でのコツや判断をまとめます。
